Bienvenue dans le Générateur de citations

Consigne

A votre tour de créer un générateur de citation Vous devez respecter les consignes suivantes :

  1. Les citations sont générées aléatoirement une par une à chaque appel du programme JavaScript
  2. Chaque citation est la combinaison d'au moins 3 morceaux de phrases (à vous de préparer ces morceaux de phrase).
  3. A défaut d'être intelligible, la phrase doit être cohérente (pas de point d'exclamation au milieu d'une phrase).

Les citations vont s'afficher ici.
